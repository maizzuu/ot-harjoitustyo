## Käynnistäminen
Ensimmäisellä käyttökerralla asenna riippuvuudet komennolla  
  
`poetry install `  
  
Seuraavaksi alusta ohjelma komennolla  
  
`poetry run invoke build`
  
Nyt ohjelman voi käynnistää komennolla  
  
`poetry run invoke start`
